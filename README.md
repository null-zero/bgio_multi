## Usage

### Installation
```sh
npm install boardgame.io
npm install --save-dev parcel-bundler
npm install esm
npm install -D tailwindcss
```
-----
### Running

#### Dev Server
```sh
npm run serve
```

#### Dev Client
```sh
npm start
```

#### Tailwind
```sh
npx tailwindcss -i ./src/input.css -o ./src/output.css --watch
```
